h2. Scripting with jark

<u>file: factorial</u>

{% highlight clojure %}
#!/Usr/bin/env jark

(ns factorial)

(defn compute [n]
   (apply * (take n (iterate inc 1))))

(println "Factorial of 10 :" (compute 10))

{% endhighlight %}

And then run it as:
<pre class="terminal"><code>$ ./factorial
=> Factorial of 10  : 3628800
</code></pre>

Jark also takes input from STDIN. One can pipe any expression to <code>jark</code>.

<pre class="terminal"><code>$ echo "(+ 2 2)" | jark -e
$ cat factorial | jark -e 
</code></pre>
